# AI-Powered Autonomous Lane Detection Robot

An autonomous robotic vehicle built using Raspberry Pi, OpenCV, TensorFlow, and Computer Vision techniques.

The system performs:

* Real-time lane detection
* Steering angle prediction
* Autonomous navigation
* Obstacle detection and avoidance

## Technologies

* Python
* OpenCV
* TensorFlow
* Raspberry Pi
* Pi Camera
* HC-SR04 Ultrasonic Sensor
* L298N Motor Driver

## Architecture

Camera → Lane Detection → Steering Prediction → Motor Controller

Ultrasonic Sensor → Obstacle Detection → Emergency Stop

## Features

* Real-time road lane detection
* AI-based steering prediction
* Obstacle avoidance
* Embedded deployment on Raspberry Pi

## Future Improvements

* Lane segmentation using U-Net
* Object detection using YOLO
* Traffic sign recognition
* Edge AI optimization using TensorFlow Lite
